# Symfony starterkit

A custom symfony starterkit.

Dependencies : 

* PHP 8
* Symfony 6
* Symfony CLI  
* Docker
* PHP_CodeSniffer

# How to use 

Clone the project : 

```
git clone git@github.com:PublicVar/symfony-starterkit.git project-name
cd project-name
docker-composer up
```

```
symfony composer install
yarn install
```

Start the server local :
```
make local
```

# Run quality tool

```
make analyse
```
